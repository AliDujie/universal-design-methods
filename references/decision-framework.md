## 一、按项目阶段选择方法

### 阶段① 规划与定义
**目标**：理解问题空间，定义项目范围
**推荐方法**：
- 必选：53.文献综述 + 80.利益相关者分析图
- 探索：39.探索性研究 + 74.次级研究 + 11.案例研究
- 深入：20.脉络访查 + 48.访谈 + 52.阶梯法

### 阶段② 探索与综合
**目标**：深入理解用户，发现设计机会
**推荐方法**：
- 必选：48.访谈 + 57.观察法 + 03.亲和图
- 田野：27.设计人种学 + 76.影形 + 59.参与观察法
- 远程：30.日记研究 + 24.文化探寻 + 37.经验取样法
- 分析：63.角色分析 + 95.客户体验历程图 + 55.心智模式图

### 阶段③ 概念生成
**目标**：激发创意，生成设计方案
**推荐方法**：
- 创意：08.头脑风暴图像组织法 + 49.KJ法 + 56.思维导图
- 共创：28.协同设计 + 61.参与性设计 + 21.创意工具包
- 原型：66.原型 + 58.平行原型 + 82.故事板
- 模拟：07.身体风暴 + 71.角色扮演 + 73.情景法

### 阶段④ 评估与细化
**目标**：测试验证，迭代优化
**推荐方法**：
- 用户测试：94.可用性测试 + 87.有声思维报告 + 68.RITE
- 专家评估：46.启发性评估 + 13.认知过程浏览
- 定量评估：01.A/B测试 + 40.眼动追踪 + 67.问卷调查
- 情感评估：29.期望值测试 + 75.语义差异法 + 54.情书与分手信

### 阶段⑤ 启动与监控
**目标**：上线监控，持续优化
**推荐方法**：
- 监控：97.网站分析 + 51.关键绩效指标 + 78.站内搜索分析
- 反馈：83.调查 + 67.问卷调查 + 22.关键事件法
- 优化：01.A/B测试 + 92.非干扰性测量

## 四、方法组合推荐（三角测量原则）

### 新产品从0到1
```
阶段①：53.文献综述 + 39.探索性研究 + 15.竞争测试
阶段②：20.脉络访查 + 48.访谈 + 03.亲和图 → 63.角色分析
阶段③：49.KJ法 + 28.协同设计 + 66.原型 + 82.故事板
阶段④：94.可用性测试 + 68.RITE + 29.期望值测试
阶段⑤：97.网站分析 + 01.A/B测试 + 83.调查
```

### 现有产品优化
```
25.用户体验审核 + 97.网站分析 + 78.站内搜索分析
→ 94.可用性测试 + 46.启发性评估
→ 01.A/B测试 + 51.关键绩效指标
```

### 服务设计项目
```
27.设计人种学 + 76.影形 + 89.试金石之旅
→ 09.商业折纸 + 95.客户体验历程图 + 72.情景描述泳道图
→ 36.经验原型 + 79.快速约会 + 99.幕后模拟
```

### 快速敏捷迭代
```
46.启发性评估（专家快速评估）
→ 68.RITE（发现即修改）
→ 01.A/B测试（数据验证）
→ 51.关键绩效指标（持续监控）
```

## 六、教练指导框架

### 对新手（0-1年经验）
- 从简单方法入手：48.访谈、57.观察法、46.启发性评估
- 使用模板和清单降低执行难度
- 重点培养：提问技巧、观察能力、数据记录习惯
- 建议先掌握10种核心方法

### 对有经验者（1-3年）
- 学习方法组合和三角测量
- 掌握综合分析技巧：03.亲和图、63.角色分析、95.客户体验历程图
- 培养洞察提炼和设计建议能力
- 学习远程研究和大规模研究方法

### 对专家/教练（3年+）
- 创新方法组合和自定义研究方案
- 培养团队研究能力
- 建立组织级研究体系
- 推动研究驱动的设计文化
- 方法论创新和最佳实践沉淀
