# References

本目录存放 Universal Design Methods 技能的方法论参考文档。

## 目录结构

| 文件 | 内容 |
|------|------|
| `decision-framework.md` | 方法选择决策框架（基于研究阶段和目标推荐方法） |
| `execution-templates.md` | 研究执行模板（访谈、观察、可用性测试模板） |
| `methods-communication.md` | 沟通阶段方法（故事板、用户旅程、利益相关者地图） |
| `methods-evaluative.md` | 评估阶段方法（可用性测试、启发式评估、A/B 测试） |
| `methods-exploration.md` | 探索阶段方法（情境访谈、日记研究、文化探针） |
| `methods-generative.md` | 生成阶段方法（头脑风暴、共创工作坊、设计工作室） |
| `methods-synthesis.md` | 综合阶段方法（亲和图、人物角色、体验历程图） |

## 使用方法

这些参考文档供 AI Agent 在执行研究任务时查阅，不需要手动维护。Agent 会根据研究阶段自动选择对应的参考文档。

## 来源

基于《Universal Methods of Design》(Bella Martin & Bruce Hanington, Rockport Publishers, 2012) 的 100 种设计研究方法体系。
