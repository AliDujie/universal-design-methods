## 二、数据采集模板

### 访谈提纲模板（方法48）
```
开场（5分钟）：
- 自我介绍和研究说明
- 知情同意和录音许可
- 破冰问题

核心问题（40-60分钟）：
1. [开放式问题] - 请描述您最近一次...的经历
   追问：当时的情境？您的感受？结果如何？
2. [行为问题] - 您通常如何...？
   追问：频率？替代方案？困难之处？
3. [态度问题] - 您对...有什么看法？
   追问：为什么？具体例子？
4. [需求问题] - 如果可以改变一件事...？
   追问：理想状态？优先级？
5. [投射问题] - 如果用一个词形容...？

收尾（5分钟）：
- 还有什么想补充的吗？
- 感谢并说明后续流程
```

## 四、研究成果沟通模板

### 研究洞察报告结构
```
1. 项目背景与目标
2. 研究方法概述
3. 关键发现（配用户语录和数据）
4. 用户角色画像
5. 体验历程图
6. 设计建议（按优先级排序）
7. 下一步行动计划
8. 附录（原始数据/详细分析）
```

### 设计建议优先级矩阵
```
| 影响力\实施难度 | 低难度 | 中难度 | 高难度 |
|----------------|--------|--------|--------|
| 高影响 | 立即做 | 计划做 | 重点项目 |
| 中影响 | 快速修复 | 排期做 | 评估后决定 |
| 低影响 | 顺手做 | 暂缓 | 不做 |
```
