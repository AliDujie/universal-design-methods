# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.8.x   | ✅ Yes             |
| 1.7.x   | ✅ Yes             |
| < 1.7   | ❌ No              |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Email the maintainers directly
3. Include a description of the vulnerability
4. Provide steps to reproduce if possible
5. We will respond within 7 days

## Security Characteristics

This skill has minimal security risk:

- **Zero external dependencies** — uses only Python standard library
- **No network access** — runs entirely offline
- **No file system access** beyond reading reference data
- **No user data collection** — all processing is local
- **Pure research toolkit** — no data exfiltration possible

## Best Practices

- Keep your Python installation up to date
- Review code before running from untrusted sources
- Use virtual environments for isolation
